# 📊 Amazon Sales Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard that analyzes sales performance, profit trends, and customer behavior.

## 🚀 Features
- Total Orders, Sales, and Profit KPIs
- Sales & Profit trend over time
- Sales distribution by category and segment
- Profit analysis by sub-category
- Shipping mode and payment method insights
- Order returns analysis

## 🛠 Tools Used
- Power BI
- Excel / CSV Dataset

## 📊 Key Insights
- West region generates highest sales
- Copiers category has highest profit
- COD is the most used payment method
- Majority of orders are not returned
